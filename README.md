## Installation

Clone the repository then create a file named `.env` and fill it out accordingly

```js
TOKEN=YOURTOKENHERE
CLIENT_ID=BOTS CLIENT ID
PREFIX=!
```

Build the project using the typescript module and start the bot using the `npm start` command
